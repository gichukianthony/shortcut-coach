# Shortcut Coach

**Shortcut Coach** tracks your VS Code commands and suggests keyboard shortcuts based on your usage.  
It helps you work faster by showing tips and a live dashboard of commands you should learn shortcuts for.

---

## Features

- Track all commands you run in VS Code
- Show tips after repeating a command multiple times
- Live **Status Bar counter** for “Shortcuts Learned Today”
- Live **Dashboard panel** showing Top 5 commands to learn
- Works automatically with built-in and extension commands

---

## Commands

- `Shortcut Coach: Open File`  
  Opens the VS Code file search and tracks usage for tips.

- `Shortcut Coach: Show Dashboard`  
  Opens a live dashboard showing your top commands to learn keyboard shortcuts for.

---

## Usage

1. Install the extension  
2. Use commands as usual in VS Code  
3. After repeating a command 5 times, a tip will appear with the keyboard shortcut  
4. The **Status Bar** shows the number of commands you have “mastered”  
5. Open the dashboard via **Command Palette → Shortcut Coach: Show Dashboard** to see top commands to learn  

---

## Status Bar

The **Status Bar** in the bottom right shows the number of commands you’ve used enough times (≥5) to have learned their shortcuts.

---

## Dashboard Panel

- Shows **Top 5 commands** you haven’t mastered yet  
- Updates **live** as you use commands  
- Displays **command name, usage count, and suggested keyboard shortcut**
